# MahlaBlog

MahlaBlog is a blogging and content management system (CMS) built with ASP.NET Core using MVC and Razor Pages. The project was developed as a practical learning experience focused on clean architecture, layered design, and modern web development practices.

## Features

* Create, edit, and delete blog posts
* Category management
* SEO-friendly URLs (Slug Generation)
* Admin panel
* Responsive user interface
* ASP.NET Core MVC
* Razor Pages
* Entity Framework Core
* SQL Server
* Layered Architecture

## Technologies

* ASP.NET Core
* C#
* Entity Framework Core
* SQL Server
* Razor Pages
* MVC Pattern
* HTML
* CSS
* JavaScript

## Project Structure

* **Presentation Layer** – User interface and controllers
* **Core Layer** – Business logic and domain models
* **Infrastructure Layer** – Data access and persistence

## Goals

This project was created to improve backend development skills and gain practical experience with ASP.NET Core, database design, and software architecture principles.
